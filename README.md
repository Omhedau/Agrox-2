# AgroX

AgroX is a platform designed to optimize agricultural practices using modern technology.

## Useful Commands

Here are some useful commands for working with this project:

- **Start the development server**:
  ```sh
  npm start
  ```

- **Build the project**:
  ```sh
  npm run build
  ```

- **Run tests**:
  ```sh
  npm test
  ```

- **Install dependencies**:
  ```sh
  npm install
  ```
